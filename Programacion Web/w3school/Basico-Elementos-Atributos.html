Este es una linea escrita con una etiqueta con mayusculas, HTML no es sensible pero por W3C es recomendable escribirlo minusculas.

La etiqueta a para referir a un link necesita href.
La etiqueta img para referir a una imagen usa src.
La etiqueta imagen necesita de especificar width=ancho, height=alto.
La etiqueta a necesita del atributo alt para mostrar como texto cuando una imagen no se puede desplegar.
El atributo style permite especificar el estilo de un elemento, ya sea color, fuente, tamaño, etc.
El atributo lang de la etiqueta html, permite definir el lenguaje de la pagina web, importante para los lectores de pantalla. Ejemplo: html lang="en-US"
El atributo title de la etiqueta p, permite desplegar un texto alternativo cuando el mouse esta sobre el elemento Ejemplo: p title="Me desplegare cuando tenga el mouse sobre mi etiqueta".
Comillas dobles o simples pueden ser usadas. Ejemplo:

  • p title='John "ShotGun" Nelson' o
  • p title="John 'ShotGun' Nelson"
